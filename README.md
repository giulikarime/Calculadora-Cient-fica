# Calculadora Científica

Uma calculadora científica desenvolvida em Python utilizando a biblioteca Tkinter, com suporte a operações matemáticas básicas, funções trigonométricas e controle por teclado.

## Funcionalidades
- Operações básicas: soma, subtração, multiplicação e divisão
- Funções científicas: seno, cosseno, tangente, raiz quadrada, logaritmo e potência
- Sistema de memória (MC, MR, M+ e M-)
- Entrada de dados via teclado e botões
- Interface gráfica simples e intuitiva

## Tecnologias utilizadas
- Python 3.11.9
- Tkinter
- Biblioteca math

## Como executar o projeto
- Acesse o projeto pelo código fonte e faça dowload
- Abra o VSCode na pasta do projeto
- Execute calculadora.py
